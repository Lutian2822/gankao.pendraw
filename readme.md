# DrawPen 手绘动画生成器

一个将静态图片自动生成「手绘过程」动画的小工具。  
使用原生 HTML / CSS / JavaScript 实现，无需打包和后端服务，适合本地或静态站点部署。

## 功能概览

- 支持「绘制目标」图片：线稿或彩色图
- 支持「手写笔」图片：可自定义握笔照片，并设置笔尖位置
- 自动识别线条，按路径驱动手部动画进行绘制
- 可选彩色填充、智能补全细节和彩色区域临时轮廓
- 支持生成视频并下载（MP4 / WebM，取决于浏览器支持）

## 快速开始

1. 克隆或下载本仓库到本地  
2. 使用浏览器直接打开 `index.html`  
   - 如果浏览器限制本地媒体录制，建议通过本地 HTTP 服务访问  
3. 按界面步骤操作：
   - 第一步：上传「绘制目标」图片（或从模板库选择）
   - 第二步：上传「手写笔」图片（或从模板库选择），根据提示点击笔尖位置
   - 第三步：调整识别阈值、笔触粗细、平滑度等细节参数
   - 第四步：设置线条绘制时长、彩色填充、智能补全等选项，点击「开始生成动画」

## 模板图片

- 绘制目标模板：`resources/drawPictures/`
  - 按 `bLine1.png`、`color1.png` 等命名可被模板库自动识别
- 手写笔模板：`resources/penPictures/`
  - 按 `pen1.png`、`pen2.png` 等命名可被模板库自动识别

只要按上述命名规则在对应文件夹中增加图片，刷新页面后即可在模板库中看到新图片。

## 兼容性说明

- 主要在现代桌面浏览器（Chrome / Edge）环境下测试
- 生成视频依赖 `MediaRecorder`，不同浏览器支持的编码格式可能不同
